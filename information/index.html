Guidance

Steps to use the tool (spider diagram) Recommendations for the user when using this tool: based on their specific requirements, e.g., if A do X, if B, do Y.

Once you define the likelihood of an unmitigated event occurring (i.e., between 0 and 5), this value applies to all the areas of people, assets, environment and reputation. The mitigated event (i.e., with safeguards) lowers the likelihood of the cause but does not lower the severity.

You can only have a 'zero' value on Severity. You cannot have a cause with a Likelihood of zero.
