Guidance

This could include the steps to use the tool - in a spider diagram format? Recommendations for the user when using this tool: based on their specific requirements, e.g., if A do X, if B, do Y.

Onece you define the likelihood of an unmitigated event occurring (i.e., between 0 and 5), this value applies to all the areas of people, assets, environment and reputation. The mitigated event (i.e., with safeguards) lowers the likelihood of the cause but does not lower the severity.

You can only have a 'zero' value on Severity. You cannot have a cause with a Likelihood of zero.

The scope of this tool: I.e., it consider equipment and chemicals (toxicity, liquid flammability). It does not consider solids flammability. It is worth noting, for the information of the user, that solids do not usually ignite, unless they are in the form of a dust.
